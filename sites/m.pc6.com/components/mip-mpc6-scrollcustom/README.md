## 属性

### sWidth

说明：款式

必选项：是

类型：Number

默认值：750

### sHeight

说明：高度

必选项：是

类型：Number

默认值：290

### sIndicatorId

说明：切换对应ID

必选项：是

类型：String

默认值：无

### sLayout

说明：布局模式

必选项：是

类型：String

默认值：fixed-height

### sNum

说明：一页分割个数

必选项：是

类型：Number

默认值：8

### sParentId

说明：外围ID名称

必选项：是

类型：String

默认值：无
