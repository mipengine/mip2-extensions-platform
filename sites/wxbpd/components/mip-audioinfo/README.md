## 说明

mip-audioinfo 用于与服务端进行数据交互并动态展示页面信息
由于官方的获取异步数据的组件对于数据格式有明确要求，并且无法满足项目需要，因此未使用官方组件

## 示例

<mip-audioinfo serverurl="" pageurl=""  section=""></mip-audioinfo>

## 属性

serverurl 获取服务端数据的地址
pageurl 页面地址，用于页面跳转
section 获取数据所需参数
