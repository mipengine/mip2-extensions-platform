## 说明

分页组件

## 示例

示例说明

```
// 代码示例
<mip-paging
  m-bind:page="page"
  m-bind:pagesize="pagesize"></mip-paging>
```

## 属性

### page

**说明**：当前页码

**必选项**：是

**默认值**：1

### pagesize

**说明**：总页数

**必选项**：是

**默认值**：0
