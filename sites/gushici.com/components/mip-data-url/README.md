类型|

## 说明

可获取访问路径中的参数来异步设置数据源

## 示例

示例说明

```
// 代码示例
<mip-data-url url="" key=""></mip-data-url>
```

## 属性

### url

**说明**：异步请求路径

**必选项**：是

**默认值**：无

### key

**说明**：设定返回的数据

**必选项**：是

**默认值**：无
