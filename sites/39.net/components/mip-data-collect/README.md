## 说明

收集用户行为信息，传输到后台

## 示例

示例说明

```
<mip-data-collect src="http://test.39.net"></mip-data-collect>
```

## 属性

### src

说明：数据传输的后台地址

必选项：是

类型：字符串

取值范围：必须是 HTTPS 的

单位：无

默认值：无
