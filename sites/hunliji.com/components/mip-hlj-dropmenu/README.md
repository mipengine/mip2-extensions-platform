## 属性

### showDownload

说明：是否显示下载 
必选项：是   
类型：字符串   
单位：无   
取值：无   
默认值：无

### menuList
说明：菜单列表 
必选项：是   
类型：字符串   
单位：无   
取值：无   
默认值：无
