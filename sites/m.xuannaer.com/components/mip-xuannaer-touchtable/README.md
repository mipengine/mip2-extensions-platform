    nb    滚动距离
**说明**：
    一个滑动的选项卡,一次滑动N个N次滑动
**必选项**：
    wrap: 'touch-wrap', // .wrap的id必须
      wrap2: 'touch-wrap2', // .wrap2的id必须
**单位**：

**默认值**：
        len: 2, 
      n: nb, // 页面一打开默认在第几页？第一页就是0
      lang: 0.5 // 每次的滑动距离0.5就是50%
