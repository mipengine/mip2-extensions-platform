## 属性

### type

说明：触发类型
必选项：是
类型：字符串
取值范围：'click'或者'load'
单位：无
默认值：load

### key

说明：事件名字
必选项：是
类型：字符串或字符串数组
取值范围：无
单位：无
默认值：无

### segmentation

说明：事件属性
必选项：否
类型：对象或对象数组
取值范围：无
单位：无
默认值：无

### href

说明：点击跳转链接
必选项：否
类型：链接类型
取值范围：无
单位：无
默认值：无
