## 说明

保险计算器功能实现，包括住房公积金、养老保险、失业保险、工伤保险、生育保险、医疗保险、养老退休金。使用该组件时请按照示例（example）文件夹中的html结构

## 属性

### type

**说明**：计算器的类型

**必选项**：是

**类型**：字符串

**取值范围**：houseFund（住房公积金）、pension（养老保险）、lostjob（失业保险）、injury（工伤保险）、birth（生育保险）、medical（医疗保险）、retire（养老退休金）
