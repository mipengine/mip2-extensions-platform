## 说明

扩展默认轮播组件，通过jsonp的方式获取数据

## 示例

url：获取jsonp的远程地址
bwidth：图片的宽度
bheight：图片的高度
defer：设置轮播间隔时间

```
<mip-bannerapi 
        url="http://yh.cszkbdfyy.com/api/1/" bwidth="1920" bheight="465" defer="5000"
>
</mip-bannerapi>
```
